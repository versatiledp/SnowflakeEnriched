Introduction¶

Snowflake Enriched is written to provide a practical guide to create a data analytics platform, including design, data collection, ingestion, transformation, and visualization.

Objectives¶

This book covers the following topics.

  • Learning Snowflake features

  • Exploring the modern big data platform

  • Working with different data formats

  • Collecting data from the public domain

  • Ingesting a variety of data formats

  • Creating database objects like tables, views, and stored procedures

  • Transforming data

  • Scheduling the data pipeline execution

  • Managing historical data loads

  • Creating data analytics insight to answer business questions

Organization of this book¶

This book is subdivided into nine topics as follows:

Introducing Snowflake covers the Snowflake architecture and data features.

Introducing data platform describes various source data files and ingesting to the internal staged area.

CSV ingestion covers the various processes involved in loading and processing CSV files.

JSON ingestion deals with ingesting and transforming JSON data files.

ORC ingestion provides insights into various aspects of loading and transformation of ORC format.

Parquet ingestion covers data ingestion and processing using Parget format files.

Multi-format ingestion describes the methods used to create lookup tables dynamically.

Workflow orchestration covers hows to build a custom workflow orchestration tool using Snowflake scripts and stored procedures.

Business insights provides insights into business use-cases identified by the user community.

Code samples¶

The data and code samples used in this book are available to download from the GitHub locations provided throughout the chapters. Follow the instructions given in each code blocks to execute data ingestion, transformation, and processing.

Prerequisites¶

The only prerequisite to follow along with this book is an Snowflake account and Google email. Readers can use Snowflake free trial services to get Snowflake account and try out all the samples from start to finish. The Google account will be needed to execute the Colab notebook.

We also assume that readers have had some working experience in database programming fields. They should be familiar in any one relational database engine and programming language.
