Introduction

Snowflake Enriched is written to provide a practical guide to create a data analytics platform, including design, data collection, ingestion, transformation, and visualization. The book provides an insight into the origin of data and how it has been transformed at all the stages of the data lifecycle. Data transformation may occur as new data is being added to existing datasets as well as when data is moved from one stage to another. This book offers detailed steps in building data platform from data engineering to presentation.

Objectives

This book covers the following topics.

  • Learning Snowflake features

  • Exploring the modern big data platform

  • Working with different data formats

  • Collecting data from the public domain

  • Ingesting a variety of data formats

  • Creating database objects like tables, views, and stored procedures

  • Transforming data

  • Scheduling the data pipeline execution

  • Managing historical data loads

  • Creating data analytics insight to answer business questions

Organization of this book

This book is subdivided into nine topics as follows:

  • Introducing Snowflake covers the Snowflake architecture and data features.

  • Introducing data platform describes various sources of data files and ingestion methods to the internal staged area.

  • CSV ingestion covers the various processes involved in loading and processing CSV files.

  • JSON ingestion deals with ingesting and transforming JSON data files.

  • ORC ingestion provides insights into various aspects of loading and transformation of ORC format files.

  • Parquet ingestion covers data ingestion and processing using Parget format files.

  • Multi-format ingestion describes the methods used to create lookup tables dynamically.

  • Workflow orchestration covers how to build a custom workflow orchestration tool using Snowflake scripts and stored procedures.

  • Business insights provides insights into business use-cases identified by the user community.

Code samples

The data files and data processing scripts used in this book are available to download from the GitHub locations provided throughout the chapters. Follow the instructions given in each code blocks to execute data ingestion, transformation, and processing.

Prerequisites

The only prerequisite to follow along with this book is a Snowflake account, SnowSQL CLI client components, and Google mail account. Readers can use Snowflake free trial services to get Snowflake account and tryout all the samples from start to finish. The Google account will be needed to execute the Colab notebook.

We also assume that readers have some working experience in database programming concepts. They should be familiar with relational database engine and programming language. For simplicity, we recommend using the data files and scripts referenced in this book.
